## [0.1.0] - 2025-01-01

### Added
- Core language: variables (`no_cap`, `lowkey`, `hits_diff`)
- Print (`yeet`, `yeet_raw`)
- Conditionals (`fr_fr` / `nah`)
- While loop (`slay`) with `ghosted` (break) and `periodt` (continue)
- Functions (`sus`) with `bet` (return) and `rizz` (call)
- Arrays with `sus_list`, `drip`, `glow_up`, `no_thoughts`
- Range with `sheesh(start, end)`
- Pattern matching with `vibe_check` / `facts:` / `cap:`
- Type casts: `based` (to number), `vibe` (to string)
- Boolean literals: `bussin` (true), `mid` (false)
- Null: `understood`
- Stdin: `ratio`
- Assert: `caught_in_4k`
- Comments: `rent_free`
- REPL mode (`--repl`)
- `.bruh` file extension enforcement
- CI/CD pipeline with GitHub Actions
- 15 integration tests
- 20+ Rust unit tests
